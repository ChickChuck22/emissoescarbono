# Grupo 5: Emissões, Carbono e Práticas ESG 🌍

Este projeto é um website interativo de duas páginas desenvolvido para apresentar estudos e slides sobre o tema de **Emissões de Gases de Efeito Estufa (GEE), Mercado de Carbono e Práticas ESG**.

O objetivo principal é fornecer uma plataforma educativa e visualmente impactante para estudantes e leigos, simplificando conceitos complexos de sustentabilidade e mudanças climáticas.

---

## 🚀 Estrutura do Projeto

O projeto foi construído utilizando tecnologias web puras (HTML5, CSS3, JavaScript), garantindo leveza e portabilidade.

### 📄 Páginas Principais

1.  **Estudos (`index.html`)**: Uma página detalhada contendo a fundamentação teórica, perguntas norteadoras respondidas, glossário simplificado e cases de sucesso (Natura e Suzano).
2.  **Slides (`slides.html`)**: Uma engine de apresentação customizada com 15 slides interativos, animações fluidas, sistema de partículas e suporte a vídeo local.

### 📁 Arquitetura de Pastas

```text
/
├── index.html          # Página principal de estudos
├── slides.html         # Engine de apresentação de slides
├── video.mp4           # Vídeo educativo sobre mercado de carbono
├── css/
│   ├── global.css      # Design System, variáveis e resets
│   ├── estudos.css     # Estilos específicos da página de estudos
│   └── slides.css      # Estilos e animações da engine de slides
├── js/
│   ├── animations.js   # Sistema de partículas e Scroll Reveal
│   └── slides.js       # Lógica de navegação e controle dos slides
└── assets/             # (Opcional) Imagens e ícones
```

## ✨ Funcionalidades dos Slides

A página de slides foi projetada para uma experiência de apresentação premium:

-   **Navegação Inteligente**: Use as setas do teclado (← →), a barra de espaço ou os controles na tela.
-   **Modo Tela Cheia**: Pressione a tecla `F` ou use o botão dedicado para uma imersão total.
-   **Progresso Visual**: Barra de progresso superior e contador dinâmico de slides.
-   **Narrativa Simplificada**: Conteúdo revisado para máxima clareza, evitando jargões técnicos excessivos.
-   **Vídeo Integrado**: Slide dedicado para demonstração visual do Mercado de Carbono.

---

## 🎨 Identidade Visual

-   **Paleta de Cores**: Baseada em tons de *Deep Slate* (`#0F172A`) e *Emerald Green* (`#10B981`), simbolizando a união entre tecnologia e natureza.
-   **Estética**: Uso de *Glassmorphism*, gradientes suaves e tipografia moderna (*Outfit* para títulos e *Inter* para textos).
-   **Interatividade**: Micro-animações em todos os botões e transições de página "fade-in/out" para slides.
